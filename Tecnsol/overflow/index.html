HTML (Hypertext Markup Language) is a standard language used to create and structure the content of web pages, defining the elements and layout that make up a webpage.

CSS (Cascading Style Sheets) is a style sheet language that works in conjunction with HTML to control the presentation and appearance of web pages, including the design, layout, and formatting aspects.

JS (JavaScript) is a programming language that enhances web pages by enabling interactivity, manipulating elements, and creating dynamic content and functionality on websites.
